# From Smiles to Machine Learning

The repo is a structured notebooks of cheminformatics concepts using RDKit.

The notebooks progress from chemical structure -> descriptors -> fingerprints -> language

## Repository Structure

Notebooks/  
    01-cheminformatics-and-rdkit-basics.ipynb  
    02-rdkit-descriptors.ipynb  
    03-rdkit-fingerprints.ipynb  
    04-smiles-tokenizer.ipynb  

## Environment
conda create -n chem python=3.10  
conda activate chem  
conda install -c conda-forge rdkit jupyterlab  
pip install numpy pandas scikit-learn
